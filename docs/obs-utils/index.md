# Welcome

Welcome to the OBS Utils documentation.

OBS Utils is a Foundry VTT module for managing the connection between Foundry and OBS Studio — it drives a streamer-friendly view of the game canvas, lets you compose actor-targeted overlays for your stream, and ties Foundry events to OBS scene switches.

## Quickstart

Start with [Getting Started](./getting-started.md) for installation and a tour of the Director window.
